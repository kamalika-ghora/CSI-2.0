Machine Learning Workshop

Machine Learning has evolved from a vague textbook expression to sophisticated algorithms, predominant in today's world without us even realising. It has become quite the trend in the 21st century and is certainly not tapering off any time soon. The Machine Learning workshop will lay emphasis on the interesting topics of Machine learning and dive deeper into object detection algorithms using OpenCV and guide in making a guided project by the end of the workshop. We will be issuing certificates by the end of the workshop and are certain that you will be instilled with a lot of curiosity and interest in Machine Learning by the end of it.

Venue : Zoom

Perks : Certificates + Vouchers
